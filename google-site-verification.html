Google Site Verification

This page is used for Google Search Console verification.

Instructions:

  1. Go to Google Search Console
  2. Add your property (marathon-pace.com)
  3. Choose "HTML file" verification method
  4. Download the verification file or copy the meta tag content
  5. Replace "YOUR_VERIFICATION_CODE_HERE" in this file with the actual verification code
  6. Upload this file to your website root directory
  7. Click "Verify" in Google Search Console

Alternative: You can also add the verification meta tag to your main index.html file instead of using this separate file.
